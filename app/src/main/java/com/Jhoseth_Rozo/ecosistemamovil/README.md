# Ecosistema Móvil - Kotlin | Unidad 1

## Descripción
Aplicación Android desarrollada en Kotlin que muestra tres paradigmas de desarrollo móvil mediante las tarjetas dinámicas.

## Cómo ejecutar el proyecto

1. Abrir Android Studio.
2. Seleccionar "Open Project".
3. Elegir la carpeta del proyecto.
4. Sincronizar Gradle.
5. Ejecutar usando el emulador.

## Problemas encontrados

- Configuración del tema AppCompat.
- Activación de la virtualización para el emulador porque daba errores al correr el emulador.
- Configuración de dependencias Gradle.

## Evidencias

- Debugger con breakpoint en forEach.
- Logcat filtrado por tag ECOSISTEMA_APP.
- Variables visibles en panel Debugger.
